✳Agentic AI — BytesAgent (IoT + LLM)
AWS Bedrock AgentCore Amazon Nova Micro Strands SDK Amazon Cognito Lambda MySQL RDS CloudFront Terraform Docker · ECR

Production-grade stateful AI agent — BytesAgent — capable of natural-language conversations, live IoT device data queries from MySQL RDS, and per-user memory across sessions. Containerised with Docker, deployed to AWS Bedrock AgentCore Runtime (zero server management), secured behind Amazon Cognito JWT authentication, with a browser-based chat UI served via CloudFront + S3. Full infrastructure as code via Terraform.
